How SSIC codes work

What you should know before picking

5-digit format

Every SSIC code is exactly 5 digits. The first 2 digits are the division (e.g., 62 = software), the next 2 narrow it (e.g., 6201 = software development), and the last digit specifies the activity (e.g., 62015 = AI/ML software).

Up to 2 codes per company

You can declare a primary SSIC code (your main business activity) and one optional secondary code. Pick codes that genuinely reflect what you'll do - misclassifying can affect grant eligibility and tax incentives.

Licensing implications

Some codes (financial services 64-66xx, F&B 56xx, education 85xx, fund management 663xx) trigger MAS, SFA, CPE or other licensing. Declaring the code does not grant the licence - you must still apply.

Free to update later

If your business pivots or expands, updating your SSIC code(s) is a free ACRA filing through BizFile+. There's no penalty for picking a less-than-perfect code at incorporation.
